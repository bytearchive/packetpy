The Options object is used inside the FlagsField class as an internal representation of the flags that can be set. In essence it is a pared down dictionary with case-insensitive keys.

Interface

Options(**kwargs)

Takes an unbounded series of name:argument pairs into the constructor that defines the initial contents of the Options object.

__getitem__(key)

Get the value stored by the key provided.

__setitem__(key, value)

Set a key:value pair.

has_key(key)

Return a boolean value based on a case-insensitive search of the keys.

keys()

Returns a list of the keys stored in the Options object. Originally provided case is retained.

values()

Returns a list of values stored in the Options object. Originally provided case is retained.
